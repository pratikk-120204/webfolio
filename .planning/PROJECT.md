# PROJECT: Webfolio Solutions

## What This Is
A premium freelance agency website for **Webfolio Solutions**, serving as a high-performance portfolio and a functional platform for client acquisition and management.

## Context
Pratik Kadole (CEO/Founder) is building a professional agency to showcase engineering marvels and offer high-end freelance services. The project aims for a "state-of-the-art" feel with modern web technologies.

## Core Value
To project a image of elite professional competence through visual excellence, seamless performance, and robust functional features (Blog, Booking, Multi-user Admin).

## Vision
To become a leading freelance agency platform that bridges the gap between high-end design and complex engineering, starting with a powerful digital presence.

## Requirements

### Active (Hypotheses)
- [ ] **Premium UI/UX**: Next.js + Framer Motion for liquid-smooth transitions and animations.
- [ ] **Multi-User Admin Panel**: Secure dashboard for Pratik and future employees/co-founders.
- [ ] **Dynamic "Storyline" Blog**: A dynamic feed for agency updates and insights, manageable via the admin panel.
- [ ] **Custom Booking & Contact**: SMTP-based lead generation with zero external service costs.
- [ ] **Case Study Gallery**: High-performance showcase of project "Marvels".
- [ ] **Supabase Integration**: PostgreSQL DB and Auth for persisting content and users.

### Out of Scope
- Paid booking integrations (e.g., Calendly Premium) — [Constraint: Budget-friendly/Free tier priority].

## Key Decisions
| Decision | Rationale | Outcome |
|----------|-----------|---------|
| Next.js (App Router) | SEO, performance, and modern developer experience. | Confirmed |
| Supabase | Robust free tier for DB, Auth, and easy integration. | Confirmed |
| Custom SMTP | Cost saving while maintaining professional functionality. | Confirmed |
| RBAC / Multi-user | To future-proof the agency for team growth. | Confirmed |
